## Social network service project

### Using...
```
- React
    - Hooks
- Firebase
    - User authentication
    - Git, google auth provider
    - Firestore Database
    - Firebase Storage
- react-router-dom
- FileReader API
```

```
- Sign in & Sign up using firebase
- Social login (Google, Github)
- CRUD chat
```
