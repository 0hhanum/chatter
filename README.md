## Social network service project

### Using...
```
- React
    - Hooks
- Firebase
    - User authentication
    - Git, google auth provider
    - Firestore Database
    - Firebase Storage
- react-router-dom
- FileReader API
- uuid (for each picture's random id)
- gh-pages deploy
```

### Features
```
- Sign in & Sign up using firebase
- Social login (Google, Github)
- CRUD chat
- Able to upload image
- Realtime update (chats, user profile ...)
```
